2. Buat database baru di phpMyAdmin bernama 'db_turnamen_futsal'.
3. Import file 'db_turnamen_futsal.sql' ke dalam database tersebut.
4. Letakkan folder proyek di dalam direktori web server (htdocs/www).
5. Akses melalui browser: localhost/KEPANITIAAN WEB/login.html
